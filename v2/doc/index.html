üéÆ How to Use

  1. Navigate to Online Tab: Go to the existing "Online" tab in PILOT navigation
  2. Find Vehicle: Locate any vehicle in the online tree
  3. Right-Click: Right-click on the vehicle to open context menu
  4. Select Option: Click "View Dashboard Panel" from the menu
  5. View Sensors: Modal window opens showing real-time sensor data
  6. Auto-Updates: Data refreshes automatically every 10 seconds
  7. Manual Refresh: Use "Refresh" button for immediate updates
  8. Close Window: Click X or press ESC to close

üõ†Ô∏è Technical Implementation

Pattern 2 Architecture

  • No Navigation Tab: Zero permanent UI footprint
  • Context Menu Integration: Adds item to existing online tree right-click menu
  • Modal Window: Ext.window.Window for sensor display
  • Tree Integration: Uses skeleton.navigation.online.online_tree
  • Auto-Cleanup: Stops refresh tasks when window closes

Context Menu Implementation

  • Safe Access: Handles both context_menu and contextmenu properties
  • Scope Binding: Uses scope: tree for access to this.record
  • Icon Required: iconCls: 'fa fa-tachometer-alt' (Font Awesome v6)
  • Preserve Existing: Only adds to menu, doesn't replace existing items

File Structure

  • V2/Module.js - Main extension entry point
  • V2/doc/index.html - This documentation file
  • V2/style.css - Optional modal window styling (optional)

üîç Usage Scenarios

When to Use V2 (Pattern 2):

  • You want minimal UI impact on existing PILOT interface
  • Sensor monitoring is occasional, not continuous
  • Users prefer context menu access
  • You need quick sensor checks without dedicated panels

When to Use V1 (Pattern 1):

  • You need continuous sensor monitoring
  • Multiple vehicles monitored simultaneously
  • Dedicated sensor monitoring workspace required
  • Users prefer permanent navigation access

‚öôÔ∏è Configuration

Extension Installation

  1. Host the V2 extension files on your web server
  2. Register the extension in PILOT admin panel with the V2 base URL
  3. Activate the extension for desired contracts/accounts
  4. The extension will add "View Dashboard Panel" to vehicle context menus
Important: Do not install both V1 and V2 simultaneously as they use the same namespace. Choose one version based on your needs.

üîÑ Real-time Updates

The modal window implements automatic data refresh with a 10-second interval. This ensures sensor values are always current while the window is open.

Performance Note: Auto-refresh automatically stops when the modal window is closed, optimizing resource usage and preventing background API calls.
